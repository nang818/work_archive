팝업/배너 2026

이커머스 메인 배너 디자인

작업 목적

신규 프로모션 배너 제작

사용 툴

Figma Photoshop

작업 연도

2026

작업 설명

사용자 시선을 끌면서도 과하지 않은 디자인. 브랜드 컬러를 활용하여 일관성 유지.

Work Process

Why

작업 배경

겨울 시즌 프로모션을 앞두고 기존 배너들이 산발적으로 제작되어 일관성이 부족했다. 마케팅팀에서 브랜드 톤앤매너를 유지하면서도 시선을 끄는 배너가 필요하다는 요청이 들어왔다.

Goal

작업 목표

시각적 임팩트는 유지하되, 브랜드 컬러와 타이포를 일관되게 사용하여 브랜드 정체성을 강화한다. 또한 배너 템플릿을 만들어 향후 시즌 배너를 빠르게 전개할 수 있도록 한다.

Approach

디자인 접근 방식

메인 비주얼은 중앙 정렬로 시선을 집중시키고, 브랜드 컬러(#2A5CFF)를 포인트로 사용했다. 타이틀은 Pretendard Bold 사용으로 가독성을 확보하고, 배경은 그라데이션을 사용해 입체감을 주었다. 정보 위계는 제목 > 부제 > CTA 버튼 순으로 명확하게 구분했다.

Output

결과물 구성

  • PC용 배너 1200x400px
  • 모바일용 배너 640x320px
  • 템플릿 컴포넌트(Figma)
  • 배너 사용 가이드 문서
Design Rule

디자인 규칙 / 가이드

  • 메인 컬러: #2A5CFF(포인트), #F5F7FA(배경)
  • 타이틀: Pretendard Bold 32px / 행간 1.4
  • 본문: Pretendard Regular 16px / 행간 1.6
  • 여백: 좌우 80px, 상하 60px (PC 기준)
  • CTA 버튼: 최소 크기 180x48px, 라운드 8px
Retrospect

아쉬웠던 점 / 개선 아이디어

CTA 버튼의 위치를 오른쪽 상단에 고정했는데, 모바일에서는 하단으로 내려야 더 접근성이 좋았을 것 같다. 다음 작업에서는 디바이스별 CTA 위치를 다르게 설정할 예정이다.
